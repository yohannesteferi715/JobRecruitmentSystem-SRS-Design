# Job Recruitment System - SRS & Design

This repository contains the **Software Requirements Specification (SRS)** and **design documents** for a Job Recruitment System, prepared as part of a software engineering course project.

## Overview

The project models a system for managing job postings, applications, and recruitment processes. It focuses on:

- Capturing functional and non-functional requirements.
- Use case analysis and diagrams.
- Object-oriented design using CRC (Class-Responsibility-Collaboration) cards.
- Class diagrams 




- **SRS/**: Contains the requirements specification (IEEE standard).
- **Design/**: Contains CRC cards, class diagrams and others .
